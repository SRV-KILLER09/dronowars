# Dron-o-War 1.0

Welcome to the official repository for the **Dron-o-War 1.0** website – the premier drone competition held at Jaypee Institute of Information Technology (JIIT).

## About the Event

**Dron-o-War 1.0** is an electrifying drone racing and innovation event designed to test the piloting skills, engineering prowess, and autonomous programming capabilities of participants. 

- **Date:** May 2-3, 2026
- **Location:** Jaypee Institute of Information Technology, Sector 128, Noida, Uttar Pradesh 201304
- **Total Prize Pool:** Rs. 8,00,000

## Featured Events

1. **FPV Racing Championship:** Fpv will be racing event with obstacles for both drone with goggles and LOS. Shortest leap time will be recorded.
2. **Payload Delivery Mission:** Payload need to drop designated payload from one spot to another while passing obstacles and points will be for landing and obstacle clearance.
3. **Autonomous Drone Mission:** Autonomous drone need to find designated hover area using cameras and hover their.
4. **RC Plane:** Design and pilot custom Radio-Controlled planes through a challenging flight course. Showcase aerodynamic efficiency, precise maneuverability, and controlled flight in confined spaces.
5. **Drone Design & Innovation:** Present a drone design or prototype demonstrating innovation, technical advancement, or a novel application.

## Tech Stack

This website is built using modern web development technologies to ensure a fast, responsive, and visually stunning experience:

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Language:** TypeScript

## Getting Started

To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Mercy036/drone-wars.git
   cd drone-wars
   ```

2. **Install the dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the application in action.
